Palantir¶

元URL: https://nbviewer.jupyter.org/github/dpeerlab/Palantir/blob/master/notebooks/Palantir_sample_notebook.ipynb

Palantirは、分化軌道に沿って細胞を整列させるアルゴリズムです。 Palantirは、低次元の表現型多様体 (phenotypic manifold)を介した一連のステップによって、幹細胞が最終分化細胞に分化する確率過程として分化をモデル化します。 Palantirは細胞状態の連続性と細胞運命決定の確率性を効果的に捉えることができます。

Normalization¶

  • Palantirは、各細胞のカウントを総分子数で正規化します。
  • データはオプションで対数変換できます。
  • いくつかのデータセットはlinear scaleでより良いシグナルを示すが, 他のデータセットは対数スケールでより強いシグナルを示す.
  • このデータセットには対数変換を用います

Running Palantir¶

  • Palantirは、early cellをおおよそ指定して実行します。

  • Palantirはterminal statesを自動的に決定しますが、termine_statesパラメータを使って指定することもできます。

  • このデータセットの開始セルは、CD34の高発現に基づいて選択されました。

In [16]:
start_cell = 'Run5_164698952452459'
pr_res = palantir.core.run_palantir(ms_data, start_cell, num_waypoints=500)

Palantir generates the following results

  • Pseudotime: Pseudo time ordering of each cell
  • Terminal state probabilities: Matrix of cells X terminal states. Each entry represents the probability of the corresponding cell reaching the respective terminal state
  • Entropy: A quantiative measure of the differentiation potential of each cell computed as the entropy of the multinomial terminal state probabilities

The terminal states in this dataset are renamed to reflect the known biology below

Gene expression trends¶

  • Palantirは Generalized Additive Models (GAMs)を使って、異なる系統に沿った遺伝子発現傾向を決定します。
  • マーカーの傾向は、以下のコマンドで決定します。これは、すべての系統のトレンドを計算します。
  • lineagesパラメータを使用して、系統のサブセットを使用することができます。
In [22]:
genes = ['CD34', 'MPO', 'GATA1', 'IRF8']
gene_trends = palantir.presults.compute_gene_trends( pr_res, imp_df.loc[:, genes])
